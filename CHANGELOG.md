# Changelog

All notable changes to the RESTLab extension will be documented in this file.

## [0.0.1] - 2024-12-11

### Added

- Initial release
- Sidebar panel with folder/request management
- Create and organize request folders
- Folder configuration (base URL, default headers)
- Full HTTP request editor (GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS)
- Custom headers with autocomplete suggestions
- Multiple content types (JSON, XML, Form URL Encoded, Form Data, Plain Text, HTML)
- Form data editor with file upload support
- Response viewer with formatted output
- Response headers display
- Status code, response time, and size indicators
- Copy response to clipboard
- Download response as file
- Copy request as cURL command
- Editable request names
- Modern gradient UI theme (Sky Blue to Indigo)
- Persistent storage across sessions
