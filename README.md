### Installation

2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "REST Lab"
4. Click Install

### Quick Start

1. **Open REST Lab**: Click the REST Lab icon in the Activity Bar (left sidebar)
2. **Create a Folder**: Click the folder icon (ğŸ“) to create your first collection
3. **Configure Folder** (Optional): Click the âš™ï¸ icon to set base URL and default headers

## ğŸ’¡ Usage Tips

- **Organize by Project**: Create separate folders for different APIs or projects
